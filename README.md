# Тестовое задание на должность DevOps-инженер


## Запуск проекта

Для начала скопируем репозиторий к себе:

`git clone https://github.com/IlyaRomanovo/effective_mobile.git`

Далее перейдем в корневую директорию проекта и запустим docker-compose.yml:

`docker compose up -d`

Готово! Проект запущен.

## Проверка результата

Для начала попробуем достучаться до порта 8080:

`curl http://localhost:8080`

Сработать не должно.

Теперь постучимся на порт 80

`curl http://localhost`

В ответ мы увидим:

`Hello from Effective Mobile!`

Чтобы посмотреть, что контейнерам задались понятные имена, можно выполнить команду:

`docker ps`

## Краткое описание работы приложения

1. Пользователь обращается по адресу `http://localhost`

2. Nginx получает запрос на порту 80

3. Nginx проксирует запрос на backend-сервис

4. Backend-сервис обрабатывает запрос на порту 8080

5. Backend-сервис возвращает ответ "Hello from Effective Mobile!"

6. Nginx передает ответ обратно пользователю
